## 2. Portfolio Description (For your personal website)

**Title**: JSON-IQ - The Intelligent JSON Developer Toolkit

**Description**:
JSON-IQ is a modern, high-performance web application designed to solve the friction developers face when handling complex data structures. Built using Next.js 16 and the Monaco Editor engine, it provides lightning-fast JSON parsing, formatting, and a custom recursive tree-view visualization capable of handling massive payloads without crashing the DOM.

Going beyond a standard utility tool, JSON-IQ integrates the Groq API (Llama 3.1) to act as an AI pair-programmer for your data. It can automatically detect syntax errors and explain them in plain English, generate comprehensive structural analytics, and instantly convert raw JSON into production-ready TypeScript interfaces. The entire application is wrapped in a bespoke, highly responsive glassmorphic UI built from the ground up with Tailwind CSS v4.

**Tags/Tech Stack**: Next.js, React, TypeScript, Tailwind CSS, Monaco Editor, Groq (Llama 3.1 LLM), Vercel.
